# env-lint
. Written in Go.
```bash
curl -LO https://github.com/javimosch/env-lint/releases/latest/download/env-lint-linux-amd64
chmod +x env-lint-linux-amd64; sudo mv env-lint-linux-amd64 /usr/local/bin/env-lint
```

This tool is a plugin for [supercli](https://github.com/javimosch/supercli) — an AI-friendly, config-driven dynamic CLI platform.
